Dr.Juke GUI
===

Графический интерфейс антивируса Dr.Juke

Запуск:
- Установить nodejs (с сайта через установщик винды)
- npm start

Зависимости:
```
npm install --save-dev electron
npm install --save-dev electron-packager
npm install --save material-design-icons
npm install --save jquery
```


Упаковка в exe:
- 32бит - npm run win-package32
- 64бит - npm run win-package64
