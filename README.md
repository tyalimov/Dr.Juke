# Dr.Juke #

#### Для сборки необходимо иметь: ####

    Visual Studio 2019
    Windows 10 SDK
    WDK 

#### Cторонние бибиотеки: ####

    boost
    CryptoPP
    curl
    Google Test
    sqlite3

#### Инструкция по сборке: ####
1. Склонировать к себе репозиторий
2. Создать в любом удобном месте директорию для загрузки зависимостей
3. Скачать архив : https://drive.google.com/open?id=1s71j2z7UBFHDh0PUvvrOU9oS_qPHOHuV
4. Распаковать архив в директорию, созданную в п.2
4. Создать переменную окружения "ExternalSdk"
5. Поместить в переменную из п.4 путь к директории, созданной в п.2
6. Открыть Dr.Juke.sln
7. Нажать Build->Rebuild Solution
